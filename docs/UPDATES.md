# Updates and Signed Releases

## How Updates Work

Zenith Codex uses Tauri's built-in updater with signed artifacts.

### Update Flow

1. **Check**: On startup, the app checks `latest.json` from GitHub Releases
2. **Download**: If a new version is available, downloads the update in the background
3. **Verify**: Validates the signature using the embedded public key
4. **Install**: Installs the update silently
5. **Relaunch**: Automatically restarts the app with the new version

### Update Endpoint

```
https://github.com/F0RLE/zenith-codex/releases/latest/download/latest.json
```

This file is automatically generated during CI builds when creating a GitHub Release from a `v*` tag.

### Manual Update Generation

If you need to generate update artifacts locally:

1. Set environment variables:
   ```bash
   export TAURI_SIGNING_PRIVATE_KEY="your_private_key_here"
   export TAURI_SIGNING_PRIVATE_KEY_PASSWORD=""
   ```

2. Build signed artifacts:
   ```bash
   cd src
   bun run app:build:signed
   ```

3. From the `src` working directory, the following files are generated in `../src-tauri/target/release/bundle/`:
   - Platform installers (.exe, .msi, .dmg, .deb, .rpm, .AppImage)
   - Signature files (*.sig)
   - `latest.json` with version info and download URLs

### Platform-Specific Updates

- **Windows**: NSIS Setup installer (.exe) supports silent background updates
- **macOS**: DMG installation; updates replace the .app bundle
- **Linux**: AppImage is self-contained and supports in-place updates

### Troubleshooting

**Updates not working?**
1. Check network connectivity to GitHub
2. Verify `latest.json` exists in the latest GitHub Release
3. Ensure the release is not a draft
4. Check console logs for signature verification errors

**Creating a new signing key** (if needed):
```bash
# Install Tauri CLI globally
cargo install tauri-cli

# Generate new keypair
tauri signer generate -w ~/.tauri/zenith-codex.key

# Public key goes in tauri.conf.json
# Private key goes in GitHub Secrets as TAURI_SIGNING_PRIVATE_KEY
```
